# Tuned Slit Drum Bank

Authority: concept/design-planning blueprint only, not fabrication-ready.

**Status:** L1 concept packet

Private-review V5 blueprint for a 12-note sheet-metal slit drum bank: twelve
small steel boxes arranged like a compact marimba, each with a laser-cut U
tongue tuned chromatically from C3 through B3. A second quieter tongue per box
adds a fifth-above shimmer after the primary pitch is validated.

## Target

- Family: idiophone / tuned slit tongue drum bank
- Scale: chromatic C3-B3, 12 boxes
- Box envelope: 200 x 100 x 80 mm each
- Material: 1.5 mm cold-rolled steel
- Play method: soft mallets
- Validation: primary pitch within +/-10 cents, sustain at least 3 seconds,
  no distracting adjacent-box cross-talk

## Packet Map

- `design.md` - acoustic and sheet-metal design rationale
