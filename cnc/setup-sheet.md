# CNC Setup Sheet

## Scope

CNC cutting is appropriate for flat box blanks, tongue slots, rack rails, and
coupons. Tuning remains manual and measurement-driven.

## Cutting Notes

- Units: mm.
- Preserve `CUT`, `BENDS`, `ETCH`, and `COUPON` layers.
- Tongue slots need smooth roots; reject jagged plasma roots.
- Use tabs only on outer waste, never on tongue tips or roots.
- Mark note labels lightly enough to finish out if desired.

## Release Check

Before releasing all twelve boxes, compare C3/F#3/B3 coupon pitch data against
the starter model and update tongue lengths if needed.
